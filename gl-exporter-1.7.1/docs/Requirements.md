# GitLab Exporter Requirements

GitLab exporter assumes the following minimum requirements are met

### Packages

Ruby version : 3.2.1

| :---  | :---------: | :---  |

| CE | 8.9.0 | Need more testing for newer versions |
| EE | 8.9.0 | Need more testing for newer versions |
